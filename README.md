
# Weather APP

  

You can view instant weather information according to the city name entered in this project. You can also view the 5-day forecast on the graph.

  Live demo: https://reacttweatherapp.netlify.app/



## Built With

  

* CRA
* ReactJS
* ApexCharts.js
* OpenWeatherMap API
